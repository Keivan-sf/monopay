# polypay.js

A node.js package for making payment transactions with different Iranian IPGs

## Documentation

- [Documentation](https://alitnk.github.io/polypay.js/)
- [مستندات فارسی](https://alitnk.github.io/polypay.js/fa/)

## Installation

For npm users:
```shell
npm install polypay
```

For yarn users:
```shell
yarn add polypay
```
